# SJ2526-CoinMetrics
**Projektbeschreibung**  
 
**Ausgangssituation**

Die Idee zu diesem Projekt entstand spontan im Unterricht. Wir möchten eine Website entwickeln, die aktuelle und übersichtliche Informationen über den Kryptowährungsmarkt bereitstellt. CoinMetrics ist eine Website, die Daten aus dem Kryptowährungsmarkt abruft, aufbereitet und benutzerfreundlich darstellt.
